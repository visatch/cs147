# Final Project for CS147 @ UCI

The idea is to build a security system that trigger by a motion sensor. It built on top of the ESP32, ESP32-CAM, ESP32-CAM-MB, Motion Sensor and it also utilizes the cloud provider (AWS) S3 to store the image.
